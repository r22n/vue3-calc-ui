
# vue3-calc-ui

the simple laptop calculator ui for vue3.

```
<template>
  <Vue3CalculatorInterface v-model="your_model_value" />
</template>

<script setup lang="ts">
import { App as Vue3CalculatorInterface } from "vue3-calc-ui";
</script>
```
